# 💼 Business Card

A responsive digital business card built using **HTML** and **CSS**. This project presents personal information in a clean, modern, and mobile-friendly layout.

---

## 📖 Overview

The Business Card project is a beginner-friendly frontend project that focuses on creating a visually appealing digital profile card. It demonstrates responsive layout design, clean typography, and organized styling using pure HTML and CSS.

---

## ✨ Features

- Responsive Design
- Clean User Interface
- Profile Information
- Social Media Icons
- Mobile Friendly
- Modern Card Layout

---

## 🛠️ Built With

- HTML5
- CSS3
